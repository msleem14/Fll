# FLL Express Logistics - موقع شركة إف إل إل إكسبريس

موقع Landing Page متكامل لشركة FLL Express Logistics المتخصصة في الخدمات اللوجستية للمطاعم والمتاجر.

## المميزات

- ✅ دعم اللغة العربية والإنجليزية مع RTL/LTR
- ✅ تصميم عصري ومتجاوب مع جميع الأجهزة
- ✅ صفحة رئيسية تعريفية
- ✅ صفحة طلب التوظيف مع نموذج متكامل
- ✅ لوحة تحكم لإدارة المحتوى
- ✅ إرسال النماذج عبر البريد الإلكتروني
- ✅ إدارة الفروع والوظائف من لوحة التحكم

## المتطلبات

- Node.js 18 أو أحدث
- npm أو yarn

## التثبيت

1. **تثبيت Node.js:**
   - حمّل من: https://nodejs.org
   - اختر النسخة LTS

2. **تثبيت المشروع:**
   ```bash
   cd fll-logistics
   npm install
   ```

3. **إعداد البريد الإلكتروني:**
   - انسخ ملف `.env.example` إلى `.env.local`
   - أضف بيانات Gmail الخاصة بك

   ```bash
   copy .env.example .env.local
   ```

4. **تشغيل المشروع:**
   ```bash
   npm run dev
   ```

5. **افتح المتصفح:**
   - الموقع: http://localhost:3000
   - لوحة التحكم: http://localhost:3000/dashboard

## لوحة التحكم

من خلال لوحة التحكم يمكنك:

1. **الإعدادات:**
   - تغيير البريد المستقبل للطلبات
   - تعديل أرقام التواصل

2. **الفروع:**
   - إضافة فروع جديدة
   - تعديل بيانات الفروع
   - تفعيل/إيقاف الفروع

3. **الوظائف:**
   - إضافة وظائف جديدة
   - تعديل الوصف والمتطلبات
   - تفعيل/إيقاف الوظائف

## إعداد البريد الإلكتروني (Gmail)

1. اذهب إلى: https://myaccount.google.com/apppasswords
2. فعّل المصادقة الثنائية إن لم تكن مفعلة
3. أنشئ كلمة مرور تطبيق جديدة
4. أضفها في ملف `.env.local`:
   ```
   EMAIL_USER=your-email@gmail.com
   EMAIL_PASS=your-16-char-app-password
   ```

## البناء للإنتاج

```bash
npm run build

```

## النشر (Deployment)

يمكن نشر الموقع على:
- Vercel (الأسهل)
- Netlify
- أي خادم يدعم Node.js

### النشر على Vercel:
1. ارفع المشروع على GitHub
2. اربطه بحساب Vercel
3. أضف متغيرات البيئة (EMAIL_USER, EMAIL_PASS)
4. انشر!
